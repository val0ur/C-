DB 파일을 활용하려 시각화하기 

주 목적은 여러 아티팩트 분석 결과를 종합해서 하나의 유의미한 결과로 만드는 것 

기능 구현보다는 어떤 아티팩트를 어떻게 조합해서 시각화로 보여줄 것인지가 핵심

Ex) timeline을 시각화한다면 사용자 행위를 나타내는 주요 아티팩트(이메일, 웹 다운로드 등)를  
     선정하고 아티팩트들을 하나로 합쳐서 시각화하는 방식 

1. 여러 아티팩트를 종합하여 의미를 찾을 수 있는 아이디어 
2. 결과 DB 파일에서 sql 쿼리로 조회하여 데이터 합치기
3. 시각화 기능 구현 - 오픈소스 시각화 API

해당 시점 10분 전 후로 timline 항목과 installed program, 검색기록을 보여준다. 

timeline은
